    public class AppFor : Command, ICommand
    Remarks

    AppFor controls loop initialisation, condition checking, and execution flow at runtime. It evaluates the loop bounds and step expressions dynamically, allowing expressions and variables to be used for loop control.

    Structural linking between the for command and its corresponding end for command is handled by the parser at compile time.
