    [TestClass]
public class IntegrationProgramTests
    Remarks

    These tests validate that multiple BOOSE language features (variables, loops, conditionals, arrays, methods, and canvas commands) operate correctly together within complete programs. Unlike unit tests, these scenarios exercise realistic execution paths across the parser, runtime, and canvas subsystems.
