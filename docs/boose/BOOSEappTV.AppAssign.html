    Remarks

    AppAssign is a core runtime command responsible for evaluating expressions and assigning their results to existing variables.

    It supports integer, real, and boolean assignments, including arithmetic, logical, and relational expressions. Expression evaluation is deferred until execution time in accordance with BOOSE’s two-pass execution model.

    This class plays a central role in program execution, as all variable updates ultimately flow through this command.
