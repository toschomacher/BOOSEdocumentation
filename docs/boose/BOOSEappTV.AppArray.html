    Syntax
    public class AppArray : Evaluation, ICommand
    Remarks

    This class provides a BOOSE-correct implementation of array variables. Arrays are declared at compile time and do not perform any action at runtime. Element access and mutation are handled exclusively via the POKE and PEEK commands.
