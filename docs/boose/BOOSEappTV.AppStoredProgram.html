    Syntax
    public class AppStoredProgram : StoredProgram, IList, ICollection, IEnumerable, ICloneable, IStoredProgram
    Remarks

    This class is the core runtime container for the BOOSEappTV interpreter. It extends BOOSE.StoredProgram to support:

    • User-defined methods with parameters and return values
    • Call stack management
    • Custom variable types (int, real, boolean)
    • Robust expression tidying and evaluation
    The class preserves BOOSEâ€™s two-pass execution model by ensuring that variable declaration occurs during parsing/compilation and that all evaluation and assignment occurs at runtime.
