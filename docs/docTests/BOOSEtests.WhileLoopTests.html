    [TestClass]
public class WhileLoopTests
    Remarks

    These tests verify correct runtime behaviour of BOOSE while loops, including:

    • Condition evaluation before loop entry
    • Multiple loop iterations
    • Nested loops
    • Nested conditional statements
    The tests ensure that loop termination conditions are honoured and that program state is updated correctly across iterations.
