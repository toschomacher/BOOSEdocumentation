    Remarks

    These tests verify that the write command correctly evaluates expressions at runtime and outputs their results to the application console. The tests cover:

    • Primitive variables (int, real, boolean)
    • Quoted string literals
    • String concatenation
    • Numeric expressions with operator precedence
    • Expressions using parentheses
    • Error handling for unknown variables
