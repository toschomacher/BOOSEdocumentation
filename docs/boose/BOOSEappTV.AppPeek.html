    Syntax
    public class AppPeek : Command, ICommand
    Remarks

    This command reads a value from an array at runtime and assigns it to a target variable. The command follows the syntax: peek varName = arrayName index. Evaluation of the index expression and assignment occur strictly at runtime in accordance with BOOSE semantics.
