    Represents an if conditional command that controls program flow based on a boolean expression.

    Syntax
    public class AppIf : Command, ICommand
    Remarks

    This command evaluates its condition at runtime and determines whether execution continues into the if body, jumps to an else block, or skips directly to the matching end if. All structural linking between if, else, and end if commands is performed by the parser.
