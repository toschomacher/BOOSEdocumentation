    public class AppReal : Evaluation, ICommand
    Remarks

    This class is a replacement for BOOSE.Real, which is sealed. It follows BOOSE-correct semantics:

    • Variable declaration occurs at compile time.
    • If an initialiser is provided, an AppAssign command is queued for runtime.
    • No evaluation is performed during compilation.
    All expression evaluation and value updates are deferred to runtime.
